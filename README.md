# `MW YTF`

## Quickstart

Install `MW YTF` with pnpm​:

```bash
  cd `skill-mw-ytf`
  pnpm install
  pnpm start
```

### Scripts

| Command      | Description                                                                                                                                                                                                                                            |
| :----------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `pnpm start` | Runs the app in development mode.                                                                                                                                                                                                                      |
| `pnpm test`  | Runs the test watcher in an interactive mode. By default, runs tests related to files changed since the last commit.                                                                                                                                   |
| `pnpm build` | Builds the skill for production to the `dist` folder. It correctly bundles React in production mode and optimizes the build for the best performance. The build is minified, and the filenames include the hashes. Your skill is ready to be deployed. |
